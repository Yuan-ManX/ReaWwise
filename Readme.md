# ReaperWwiseTransfer

## Pre-requisites
- cmake 3.22.0
- git
- Wwise (with SDK source)

## Building the project
- git clone {repo-url}
- cd ReaperWwiseTransfer
- git submodule update --init
- mkdir build
- cd build
- cmake ..
